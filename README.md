VSCode or another IDE required  
Python 3.11 required  
Open main.py and copy these commands into the terminal:  
pip install pywin32 pynput pyinstaller  
pyinstaller --onefile --windowed --name "DotControlPanel" main.py  
You will get DotControlPanel.exe in 'dist\' folder   
One of the app's modes may reduce motion sickness, but it's not certain
